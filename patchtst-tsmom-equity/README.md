# Forecasting Time-Series Momentum in U.S. Equities with PatchTST

**STAT0035 Statistics Project — University College London, 2025–26**

This repository contains the complete codebase for the STAT0035 project report *Forecasting Time-Series Momentum in U.S. Equities with PatchTST*.

## Overview

The project applies the Patch Time Series Transformer (PatchTST) architecture to volatility-scaled daily log returns for 100 U.S. equities across three market-capitalisation tiers (1990–2024). Nine configurations spanning three lookback windows and three forecast horizons are trained and evaluated out of sample against Random Walk, AR(1), and Historical Mean benchmarks.

## Reproduction

### Prerequisites

- Python 3.10+
- NVIDIA GPU recommended (trained on RTX 4070)
- WRDS account for CRSP data access

### Setup

```bash
git clone https://github.com/YOUR_USERNAME/patchtst-tsmom-equity.git
cd patchtst-tsmom-equity
pip install -r requirements.txt
```

Set your WRDS credentials as an environment variable:

```bash
export WRDS_USERNAME=your_username
```

### Execution Order

Run the scripts in this order:

1. **`data/cohort_1990.py`** — Constructs the 1990 cohort (outputs `cohort_1990_stocks.csv`)
2. **`data/cohort_2024.py`** — Constructs the 2024 cohort (outputs `cohort_2024_stocks.csv`)
3. **`data/preprocessing.py`** — Pulls CRSP returns, applies volatility scaling (outputs `preprocessed_data.pkl`)
4. **`analysis/eda.py`** — Generates all Section IV figures and tables
5. **`models/train.py`** — Trains all 9 PatchTST models (outputs to `models_v2/`)
6. **`models/evaluate.py`** — Evaluates against benchmarks, runs factor regressions (outputs to `results_v2/`)
7. **`analysis/regime.py`** — Regime-conditional analysis for L252_h21
8. **`models/ablation.py`** — Ablation study and multi-horizon ensemble (outputs to `results_ablation/`)

### Data

Daily return data is sourced from the CRSP Daily Stock File via WRDS. A WRDS institutional subscription is required. The cohort CSVs and preprocessed pickle file are intermediate outputs generated by the data scripts.

## Key Results

- PatchTST does not outperform naive benchmarks in point-forecast accuracy or directional accuracy
- At the classical TSMOM horizon (L=252, h=21), the model achieves a mean IC of 0.031 with 38% of assets significant after BH correction
- No statistically significant alpha after Carhart four-factor adjustment
- Performance concentrates in trending regimes (IC = 0.172 during post-COVID recovery) and deteriorates during regime transitions
- Ablation study confirms the binding constraint is data informativeness, not model capacity
